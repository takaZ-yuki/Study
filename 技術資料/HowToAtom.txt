
■Atom ショートカットキー

・対応する括弧、タグに飛ぶ
→ Ctrl + m

・指定行に飛ぶ
→ Ctrl + g（桁番号を指定する場合 例） 20:10）

・パネル分割
→ Ctrl + k + 矢印

・現在の行の移動
→ Ctrl + 矢印

・現在の行を選択
→ Ctrl + l

・現在の行の削除
→ Ctrl + Shift + k

・現在の行の複製
→ Ctrl + Shift + d

・ファイル形式を選択
→ Ctrl + Shift + l

・選択行のコメントアウト
→ Ctrl + /

・前の行に挿入
→ Ctrl + Shift + Enter

・検索
→ Ctrl + f

・検索　次の行
→ F3

・検索　前の行
→ Shift + F3

・シンボルの検索
→ Ctrl + r

・ファイル検索
→ Ctrl + t

・開いているタブから検索
→ Ctrl + b

・開いているタブから検索
→ Ctrl + b

・大文字化
→ Ctrl + u

・コマンドパレット
→ Ctrl + Shift + p

・プロジェクトフォルダの追加
→ Ctrl + Alt + o

・環境設定
→ Ctrl + ,

・ウインドウを閉じる
→ Ctrl + w

・折りたたむ
→ Ctrl + Alt + [

・折りたたみを戻す
→ Ctrl + Alt + ]

・単語の終わりまたは初めまで移動
→ Ctrl + 横矢印

・単語の終わりまたは初めまで選択
→ Ctrl + Shift + 横矢印]

・ツリービューを開く/閉じる
→ Ctrl + \

・ソースを整形（Atom Beautify）
→ Ctrl + Alt + B
